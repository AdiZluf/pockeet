# Contributing

Thanks for helping build Pockeet.

## Before you start

1. Read [docs/README.md](docs/README.md) and [docs/product/mvp-scope.md](docs/product/mvp-scope.md).
2. Follow [docs/ai/cursor-rules.md](docs/ai/cursor-rules.md) for code and UI conventions.
3. For your role, see [docs/ai/onboarding.md](docs/ai/onboarding.md).

## Pull requests

- Keep changes scoped to the MVP loop unless discussed.
- UI PRs: verify **EN + HE RTL** and largest dynamic type where layout changes.
- Token changes: confirm contrast per [docs/accessibility/README.md](docs/accessibility/README.md).
- Update docs in the same PR if behavior diverges from specs.

## Documentation

Specs live under `docs/` — update the owning file rather than duplicating in PR descriptions.
